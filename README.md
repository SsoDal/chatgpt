# 📊 트레이딩 판단 보조 시스템

## 🔥 핵심 철학

- ❌ **자동매매 없음**
- ✅ **사람이 최종 결정**
- ✅ **정확한 근거 + 수치 제공**

## 🧠 시스템 구조

### 1️⃣ 데이터 레이어
- 뉴스 수집 (네이버/다음/연합/해외)
- 주가 (현재가)
- 거래량 (현재 + 평균 대비)

### 2️⃣ 분석 레이어
- 뉴스 → 종목 추출 (AI)
- 뉴스 감성 점수
- 거래량 필터
- 가격 위치 (고점/저점)

### 3️⃣ 전략 레이어
- 단타 / 스윙 분리
- 손절 / 익절 자동 계산
- 위험도 평가

### 4️⃣ 출력 레이어
- **텔레그램**: 요약 + 핵심 시그널
- **웹**: 상세 데이터 + 분석

## 🚀 설치 및 실행

### 1. 환경변수 설정

```bash
export TELEGRAM_TOKEN="your_telegram_bot_token"
export TELEGRAM_CHAT_ID="your_chat_id"
export OPENAI_API_KEY="your_openai_api_key"
```

### 2. 패키지 설치

```bash
pip install -r requirements.txt
```

### 3. 실행

#### 메인 시스템
```bash
python main.py
```

#### 웹 대시보드
```bash
cd web
python app.py
```

## 📁 파일 구조

```
trading-assistant/
├── analyzer.py          # 뉴스 분석 + 종목 추출

```

## 🎯 주요 기능

### 📊 뉴스 감성 분석
- **긍정 키워드**: 수주, 계약, 호재, 급등, 성장, 흑자
- **부정 키워드**: 적자, 하락, 악재, 손실, 규제

### 📈 전략 분류
- **🔥 단타 강**: 점수 3점 이상 + 거래량 500만주 이상
- **⚡ 단타**: 점수 2점 이상
- **📊 스윙**: 점수 1점 이상
- **관망**: 그 외

### 💰 손절/익절 계산
- **손절**: 현재가 × 0.97 (3% 손절)
- **익절**:
  - 점수 3점 이상: 현재가 × 1.07 (7% 익절)
  - 점수 2점 이상: 현재가 × 1.05 (5% 익절)
  - 그 외: 현재가 × 1.03 (3% 익절)
